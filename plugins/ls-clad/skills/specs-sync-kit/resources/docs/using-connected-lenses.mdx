# Using Connected Lenses

> Upstream docs for figures and diagrams: [Using Connected Lenses](https://developers.snap.com/spectacles/about-spectacles-features/connected-lenses/using-connected-lenses).

SnapOS provides multiple entry points for Connected Lenses on Spectacles. A Connected Lens session can be created or joined from Lens Explorer.

## Creating a New Session

> See [Creating a New Session](https://developers.snap.com/spectacles/about-spectacles-features/connected-lenses/using-connected-lenses#creating-a-new-session) in the upstream docs for diagrams.

Connected Lenses are available under the Multiplayer section of Lens Explorer. To start a Connected Lens session, follow these steps:

1. Select the **Multiplayer** category in Lens Explorer.
1. Launch your chosen Lens.
1. Choose **Multiplayer** in the Lens UI to open the system dialog for selecting a session.
1. Select **Start New** to create a new session.

## Joining an Existing Session

There are two ways to join an active Connected Lens session from Lens Explorer:

1. Multiplayer category
1. Active Nearby category

### Multiplayer Category

> See [Multiplayer Category](https://developers.snap.com/spectacles/about-spectacles-features/connected-lenses/using-connected-lenses#multiplayer-category) in the upstream docs for diagrams.

1. Select the **Multiplayer** category in Lens Explorer.
1. Launch the same Lens as the person you want to join.
1. Choose **Multiplayer** in the Lens UI to open the system dialog for selecting a session.
1. Select the session that has the name(s) and Bitmoji(s) of those you would like to join.

### Active Nearby Category

> See [Active Nearby Category](https://developers.snap.com/spectacles/about-spectacles-features/connected-lenses/using-connected-lenses#active-nearby-category) in the upstream docs for diagrams.

1. Select the **Active Nearby** category in Lens Explorer. Lens Explorer will show tiles of Connected Lenses sessions that are active around you.
1. Select the tile for a Connected Lens session you would like to join.

## Establishing a Shared Coordinate Space

> See [Establishing a Shared Coordinate Space](https://developers.snap.com/spectacles/about-spectacles-features/connected-lenses/using-connected-lenses#establishing-a-shared-coordinate-space) in the upstream docs for diagrams.

Connected Lenses on Spectacles are colocated. The first user to join the Lens establishes the shared coordinate space automatically. Subsequent players are guided to look at a friend who has already colocated to align their space with theirs.

Ensure successful colocation by:

- Facing your friend and looking at them head-on
- Standing within 2 meters of the friend you are looking at
- Using adequate lighting to make features of your space detectable

Once you colocate successfully, Connected Lenses session setup is complete!
