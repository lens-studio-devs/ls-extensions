# Payload and Rate Limits

Connected Lenses need to stay within the payload and message rate limits when synchronizing data. The Lens Studio Logger will show an error if limits are exceeded, and the session may disconnect.

### Payload Limit

The maximum payload limit for a message is 100 KB in both Lens Studio and on Spectacles.

### Rate Limit

The rate limit for messages is 350 messages per 5 second window in both Lens Studio and on Spectacles.
