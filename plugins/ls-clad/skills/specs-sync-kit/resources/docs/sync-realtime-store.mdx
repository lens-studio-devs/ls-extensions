# Sync Realtime Store

> Upstream docs for figures and diagrams: [Sync Realtime Store](https://developers.snap.com/spectacles/spectacles-frameworks/spectacles-sync-kit/features/sync-realtime-store).

The SyncRealtimeStore component is meant to be a very simple interface for a SyncEntity and its Realtime Store. It does not do any behaviors on its own, so it can be used just for storing and retrieving synced values.
