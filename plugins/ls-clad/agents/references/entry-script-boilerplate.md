# Self-contained entry-script boilerplate (full)

Reference for `specs-experience-builder` Essential Pattern #7. Consulted at Phase 2f write time. The full implementation below includes the `setColliderDebugAll` walker (Hard Rule 6.7) and the wrapper-pattern `buildScene()` body (Hard Rule 6.2). Both are load-bearing; the always-loaded agent body carries only the class shell + `@input` declarations + `onAwake()` stub.

```typescript
import { GameExperienceUI } from "./GameExperienceUI";  // class generated by /specs-build-ui (Phase 2d)
const boardPrefab = requireAsset('../GeneratedMeshes/board.glb') as ObjectPrefab;

@component
export class GameExperience extends BaseScriptComponent {
    // Inspector toggle: flip ON to render every Physics.ColliderComponent /
    // BodyComponent in the scene as a wireframe — for diagnosing hit-zone
    // mismatches against the visual mesh. Takes effect on preview restart.
    @input
    debugColliders: boolean = false;

    // Wired by the bootstrap to the UI panel's ScriptComponent (Hard Rule 3).
    // Typed as the UI class directly — Lens Studio resolves the wired
    // ScriptComponent to the underlying class instance at runtime, so no
    // `.getScript()` and no cast.
    @input
    uiHud!: GameExperienceUI;

    private score = 0;

    onAwake() {
        this.buildScene();
        this.uiHud.onStart.add(() => this.startGame());
        if (this.debugColliders) {
            this.setColliderDebugAll(this.getSceneObject(), true);
        }
    }

    private buildScene(): void {
        const root = this.getSceneObject();

        // Wrapper pattern (Hard Rule 6): collider + Interactable on unit-scale wrapper
        const boardWrapper = global.scene.createSceneObject('BoardWrapper');
        boardWrapper.setParent(root);
        boardWrapper.getTransform().setLocalPosition(new vec3(0, 0, -110));

        // Default scale (100) preserves Blender-authored size — see GLB scale rule above
        const board = boardPrefab.instantiate(boardWrapper);

        // Collider on wrapper; size = aabb_cm recorded in Phase 2b (filled in Phase 2g)
        const collider = boardWrapper.createComponent('Physics.ColliderComponent') as ColliderComponent;
        const box = Shape.createBoxShape();
        box.size = BOARD_AABB;  // TODO: verify with aabb_cm
        collider.shape = box;
        boardWrapper.createComponent(Interactable.getTypeName());
    }

    private startGame(): void {
        this.score = 0;
        this.uiHud.setScore(this.score);   // push state into UI — never createComponent('Component.Text')
    }

    // Walks the scene tree from `obj` and toggles debugDrawEnabled on every
    // Physics.ColliderComponent and Physics.BodyComponent. Drives the
    // `debugColliders` @input above. BodyComponent is checked separately
    // because getComponent('Physics.ColliderComponent') does not return
    // BodyComponents even though BodyComponent extends ColliderComponent.
    private setColliderDebugAll(obj: SceneObject, enable: boolean): void {
        const c = obj.getComponent('Physics.ColliderComponent') as ColliderComponent | null;
        if (c) c.debugDrawEnabled = enable;
        const b = obj.getComponent('Physics.BodyComponent') as ColliderComponent | null;
        if (b) b.debugDrawEnabled = enable;
        for (let i = 0; i < obj.getChildrenCount(); i++) {
            this.setColliderDebugAll(obj.getChild(i), enable);
        }
    }
}
```
